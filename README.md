🎬 LuraFlix
O LuraFlix é uma plataforma de gerenciamento de vídeos inspirada no layout de serviços de streaming. O projeto permite a organização de conteúdos por categorias personalizadas, oferecendo uma interface limpa, moderna e totalmente funcional.

Este projeto foi desenvolvido para colocar em prática conceitos de geração de código com AI, focando em componentização, roteamento e estilização dinâmica.

🚀 Funcionalidades
Organização por Categorias: Vídeos agrupados por temas (Front-end, Back-end, Inovação, etc).

Cadastro de Vídeos: Formulário estruturado para adição de novos conteúdos.

Navegação Fluida: Uso de Single Page Application (SPA) para transições sem recarregamento.

Design Responsivo: Adaptado para diferentes tamanhos de tela.

Estilização Dinâmica: Cores de categorias que se refletem nos componentes.

🛠️ Tecnologias Utilizadas
O projeto foi construído com:

VsCode - Com AI gemini

Styled Components - CSS-in-JS para estilização modular e dinâmica.

JSON Server (Simulação de API) - Para persistência de dados durante o desenvolvimento.

📦 Como rodar o projeto
Se tu quiser testar a máquina na sua conta local, segue o passo a passo:

Clone o repositório:

Bash
git clone https://github.com/flatauchen/LuraFlix.git
Entre na pasta do projeto:

Bash
cd LuraFlix
Instale as dependências:

Bash
npm install
Inicie o servidor de desenvolvimento:

Bash
npm start
O projeto abrirá automaticamente no seu navegador em http://localhost:3000.
